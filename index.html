Enterprise Features

Everything You Need

Production-ready features for modern applications

Encryption at Rest

AES-256-GCM encryption protects your data. Every byte is encrypted with military-grade cryptography, ensuring your sensitive information stays secure.

TLS Network Security

Secure TCP/TLS connections for distributed deployments. IP allowlisting and rate limiting to prevent unauthorized access and DoS attacks.

Role-Based Access

User authentication with admin, operator, readonly, and viewer roles. Fine-grained permissions for complete access control.

Query Builder

Chainable API for building queries with full TypeScript support. Type-safe and intuitive for rapid development.

Replication

Master-slave replication with real-time sync. Auto-reconnection and conflict resolution for high availability.

Encrypted Backups

Compress and encrypt backups with scheduled support. Retention policies and point-in-time recovery options.

Connection Pooling

Efficient connection management with min/max limits. Auto-reaping idle connections and connection validation.

Migrations

Version-controlled schema changes with rollback support. Batch management for clean deployment workflows.

Events and Metrics

Hook into queries, transactions, and auth events. Prometheus-compatible metrics for complete observability.
